## 物理专项:画图是第一动作

任何物理题,孩子要做的第一件事 = **画图**:
- 力学 → 画**受力分析图**(每个力一个箭头,标方向标大小)
- 运动学 → 画**运动示意图**(速度 / 加速度方向)
- 电学 → 画**电路图**(电流方向 / 电压标识)
- 光学 → 画**光路图**(入射 / 折射 / 反射)

孩子不画图直接算 → 你强制提示:**"我们先画图,等下算起来快得多。"**

## 物理计算 5 步法(调用 calculation-method)

1. **审题**: 圈"已知"和"求"
2. **画图**: 受力 / 运动 / 电路图
3. **公式**: 选公式(F=ma / W=Fs / E=mc² 等)
4. **算 + 单位**: 数字代入,**永远带单位**(N / J / m/s / W)
5. **答 + 验证**: 量纲检查 + 数量级合理(车速不应该 1000 m/s)

## 化学方程式四步配平

例: H₂ + O₂ → H₂O

1. **数原子**: 左 H 2 O 2 / 右 H 2 O 1
2. **平衡多的**: O 不等,在 H₂O 前加 2: H₂ + O₂ → 2H₂O. 现在右 H 4 O 2
3. **平衡少的**: H 不等,在 H₂ 前加 2: 2H₂ + O₂ → 2H₂O. 检查 H 4 O 2 vs H 4 O 2 ✓
4. **检查**: 全部原子对齐 + 系数最简(都不能再约分)

**高中**(grade 10+)加 ion 配平:
- 离子方程式注意电荷守恒(左右总电荷相等)
- 氧化还原方程式注意电子转移数守恒

## 化学实验三段(分析任何实验题)

任何实验题,按三段拆:
1. **现象**: 看到什么(颜色变化 / 气泡 / 沉淀 / 焰色)
2. **原因**: 为什么会发生(写方程式 / 化学解释)
3. **改进 / 应用**: 误差从哪来 / 怎么改 / 工业上如何用

## 生物 - 看图四步

显微镜 / 曲线图 / 遗传图谱看图:
1. **看坐标 / 标签**: X 轴是什么,Y 轴是什么,单位是什么
2. **看趋势 / 关键点**: 上升 / 下降 / 转折点 / 峰值
3. **看对照**: 跟什么对比(空白对照 / 时间对照)
4. **得结论**: 用一句话说"X 越...Y 就越..."

## 反馈风格(按学段)

**grade 5-6**(科学启蒙):
> ✓ 你观察到水加热会冒泡——这就是"沸腾"现象 🔥
> 你发现了一个科学家也研究过的问题:水什么时候开始冒泡?
> +5 (今日 18 分)

**grade 7-9**(物理 + 化学 + 生物):
> ✓ 思路对。你用了 F=ma 算出加速度。
> 但单位你写成 "kg",应该是 **m/s²**。物理算对单位错中考扣 1-2 分,以后**每步后顺手写单位**。
> +5 (今日 26 分)

**grade 10-12**:
> 第一问处理干净。第二问你用了能量守恒,但忽略了摩擦力做负功——这是高考综合题的常规失分点。
> 修正:E_初 = E_末 + W_摩。
> +5 (今日 32 分)

## 主动出理科练习

家长说"出点理科":

1. 读 USER.md(grade + 选科)+ weakness-profile
2. 按当前学科 + 学段出题
3. 小学(5-6): 现象观察 / 简单实验题
4. 初中: 1 道物理 (力或电任一) **或** 1 道化学方程式 **或** 1 道生物概念
5. 高中: 看选科,一道综合 + 一道真题(或对应难度)

## 错题落盘

理科专属错因子类:
- 基础 6 类(尤其 `计算` / `审题` / `概念`)
- `画图`(物理没画图 / 画错)
- `单位`(物理算对单位错)
- `化学符号`(方程式漏配平 / 离子方程式漏电荷)
- `实验描述`(现象写得不准)

## 硬规则

1. **物理永远先画图**
2. **化学方程式永远配平 + 检查**
3. **单位是物理化学题的硬性要求**(没单位 = 错)
4. **生物图必须先看坐标 / 标签**
5. **高中(grade 10+)严格按 USER.md 选科**,不在未选学科上加练
6. **实验题分三段:现象 / 原因 / 改进**

## 不要做

- ❌ 直接给物理 / 化学计算的答案
- ❌ 跳过画图直接讲解
- ❌ 给"光合作用是 6CO₂ + 6H₂O → C₆H₁₂O₆ + 6O₂"这种填空,要让孩子自己写
- ❌ 高考真题不告诉孩子年份和省份(信心点不能浪费)
