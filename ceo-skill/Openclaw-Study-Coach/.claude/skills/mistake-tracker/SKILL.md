# 错题追踪 Skill

## 什么时候触发

- 孩子做错一道题(homework-intake 流程中自动触发)
- 家长或孩子说"加入错题本"、"记下这题"、"这题以后再练"
- 老师批改后家长发来错题

## 输入(任选)

- 错题原文(文字 / 图片)
- 学科(可选,不填由内容判断)
- 孩子当时的答案
- 家长 / 自己的观察:"是审题问题" / "不会算" / "不会"

## 必须包含的错题卡字段

| 字段 | 说明 | 必填 |
|---|---|---|
| 学科 | 数学 / 语文 / 英语 | ✓ |
| 知识点 | 简短,2-6 字,比如"分数加减"、"形近字" | ✓ |
| 日期 | YYYY-MM-DD | ✓ |
| 错因分类 | 必从 AGENTS.md 的 6 类中选 1-2 个:**审题/计算/概念/方法/书写/习惯**。**禁止用"粗心"**。 | ✓ |
| 正确思路 | ≤ 6 句话 | ✓ |
| 一句纠错要点 | 孩子能背出来的一句话 | ✓ |
| 类题建议 | 2 道(题目描述即可,不一定要标准答案) | ✓ |
| 回流日期 | 明天 / 三天后 / 一周后(根据错因严重程度) | ✓ |
| 状态 | pending / done | ✓(初始 pending) |

## 回流日期决策表

| 错因 | 建议回流 |
|---|---|
| 概念 | 明天 + 3 天后 + 7 天后(三次回流) |
| 方法 | 3 天后 + 7 天后 |
| 审题 / 计算 | 3 天后 |
| 书写 / 习惯 | 不进队列,但写错题卡,周报里看趋势 |

## 输出格式

```markdown
### 错题卡

**学科**: 数学
**知识点**: 分数加减
**日期**: 2026-05-22
**错因分类**: [审题]

#### 题目
1/2 + 1/3 = ?

#### 你的答案
2/5

#### 正确思路
1. 分母不一样,先通分
2. 1/2 = 3/6,1/3 = 2/6
3. 3/6 + 2/6 = 5/6

#### 一句纠错要点
"分母不一样,先通分再加"——把这句记住就够了。

#### 类题(后面会回来练)
1. 1/4 + 1/2
2. 2/3 + 1/6

#### 下次回流
3 天后(2026-05-25)

---

### 给家长的一句话(可选)

今天数学第 4 关错在分母没通分,这是这周第二次。不是粗心,是分数加减的概念还没稳。建议明后两天少看新内容,把这一类再过两遍。

---

### 落盘情况

- ✅ 错题卡: `data/mistakes/subjects/数学/2026-05-22_分数加减_mistake.md`
- ✅ 加入队列: `data/mistakes/review_queue/2026-05-22_queue.md`
```

## 硬规则

1. **错因分类禁止用"粗心"**。永远拆成审题 / 计算 / 概念 / 方法 / 书写 / 习惯。
2. **同一知识点的错题归到同一目录**:`data/mistakes/subjects/<学科>/<知识点>/...` 也可以(如果一个知识点错得多)。
3. **类题不要超过 2 道**。多了孩子直接抗拒。
4. **必须落 2 个文件**:学科归档 + 当日队列。
5. **每次新增错题,trends.csv 的 `mistakes_new` 字段会被 daily-report 累加**——你不用直接动 csv,但要在反馈里告诉 daily-report skill 今天新增了几条。

## 同一道题再错(回流后还错)

新建一份 `_v2` 后缀的错题卡,标记 `状态: 复发`。回流日期推迟到 7 天后(再给一次机会)。同时在反馈里加一句给家长:"⚠️ 这道题是 X 天前的回锅题,还是错。建议晚上单独讲 3 分钟。"

## 当孩子错的特别多(一天 > 8 题)

不要每道都生成完整卡片。**优先处理 3 道最重要的**(同一知识点 / 相同错因 / 影响后面学新内容的)。剩下的合并成一张"轻错题清单",只记题目+错因,不写正确思路。原因:小学生情绪管理 > 完整记录。

## 当题目是英语 / 语文

- 英语错题:用英文原题 + 中文解释正确思路。一句纠错要点用中文。
- 语文错题:如果是字词,在"题目"区直接列易错字(比如"掩(yǎn)耳盗铃 — 掩 ≠ 淹");如果是阅读理解,把原文段落+问题+答案都贴出来。

## 不要触发

- 孩子写错了一个字但立刻自己改对 → 不触发,这不算错题
- 同一节课内连续 3 次错同一道题 → 触发一次,后两次合并到同一张卡(不要生成 3 张)
- 孩子明显在乱按 / 在玩 → 不触发,可能是注意力问题不是学习问题
