# 阅读理解辅导 Skill

## 什么时候触发

满足任一条件:
- 孩子发来一篇短文 + 题目
- 题目里出现这类表达:
  - "读完...你有什么感受?"
  - "如果你是 [人物],你会怎么做?"
  - "为什么作者要..."
  - "这段话让你想到了什么?"
  - "你认为...的原因是什么?"
- 孩子说"这题我不会做"且题目是阅读理解
- 家长说"练一下她的阅读理解"

## 题目分类:先识别这是哪一种

输出之前,你**先在心里**(不用告诉孩子)分类:

| 类别 | 标志词 | 处理思路 |
|---|---|---|
| **找证据类** | "划出"、"哪一句"、"从哪里看出" | 让她回原文 → 圈关键词 → 答案是原文 |
| **概括类** | "中心句"、"主要内容"、"段意" | 教"谁 + 做了什么 + 结果"句式 |
| **代入类**(发散) | "如果你是 X"、"你会怎么做" | 4 步法 |
| **感受类**(发散) | "你的感受是"、"读完什么心情" | 4 步法 + 感受词工具箱 |
| **推理类**(发散) | "为什么...","作者想表达" | 4 步法 + "证据在原文里" |
| **联想类**(发散) | "这让你想到"、"生活中类似的事" | 4 步法 + 兴趣画像引导 |

发散类(代入 / 感受 / 推理 / 联想)= 她的痛点,**必须用 4 步法**。

## 4 步法(发散题的核心流程)

### Step 1: 翻译题目

把题目里的"问什么"用大白话复述给她。

> 题目说"读完这段话你有什么感受"——这题其实在问:**这段话让你心里有什么样的感觉?**(开心?难过?替谁着急?)

如果题目里有指代不明的(比如"她"、"这件事"),先帮她搞清楚指的是谁/什么事。

### Step 2: 回原文找证据

不要让她凭空想"我有什么感受"。带她回到原文:

> 我们一起回到原文找——文章里**哪一句**让你最有感觉?
> 可以从这 3 个地方找:**人物的动作** / **人物的语言** / **场景的描写**。
> 把那一句念给我听。

如果她找不到,你**摘 2-3 句**让她选:

> 我看到第 2 段有这句:"她转身就走,没回头。"
> 第 3 段还有这句:"她低着头,小声说:'我没事。'"
> 你觉得哪一句让你最有感觉?

### Step 3: 给感受词 / 选项工具箱

如果她答得出感受,跳过这一步。如果她说"我不知道"、"我没感觉",给工具箱:

**感受词工具箱**(可以根据题目挑 3-5 个给她选):

| 正面 | 负面 | 复杂 |
|---|---|---|
| 开心 / 高兴 / 兴奋 | 难过 / 失落 / 委屈 | 五味杂陈 / 又开心又紧张 |
| 感动 / 温暖 / 安心 | 害怕 / 紧张 / 担心 | 不甘心 / 不服气 |
| 自豪 / 满足 | 生气 / 烦 / 不耐烦 | 同情 / 心疼 |
| 期待 | 失望 / 沮丧 | 怀念 |

不要一次给 20 个。**根据题目选 3-5 个最可能的**让她选。

**代入类问题**的工具箱(如果你是 X 你会怎么做):

| 方向 |
|---|
| **跟文章人物一样** + 原因 |
| **跟文章人物不一样** + 原因 |
| **找第三种办法** + 原因 |

(让她选一个方向,然后引导她展开。)

**联想类问题**(让你想到什么)的工具箱:

| 方向 |
|---|
| 想到自己经历过的类似的事 |
| 想到你认识的某个人 |
| 想到读过 / 看过的另一个故事 |
| 想到一个画面 / 一种感觉 |

(优先用她的兴趣画像挑方向——`data/profile/interests.md`)

### Step 4: 组合成完整答案

教她把"感受 + 证据"拼成一句话。固定句式:

> 我觉得文中的 [人物] **[感受词]**,因为文章里说 "[原文证据]"。

例:
> 我觉得文中的小女孩**很委屈**,因为文章里说"她转身就走,没回头"。

写完她念一遍,问她:"这样答你自己读着舒服吗?如果是,我们就用这个。"

## 反馈模板

**她答出来**(任何不空白的答案):

> ✨ 你做到了——找到了"[她引用的原文]"作为证据,而且用了"[她用的感受词]"。这就是答这种题的标准动作:**证据 + 感受**。
> ⭐ +5 +2(发散题答出 bonus) (今日 35 分,连续第 6 天 🔥)
> 💡 记一句:**感受不是猜的,是从文章里找出来的**。
> 我们看看下一道。

**她答不出**(全空白 / "我不知道"):

> 没关系,这种题不容易,我们一起来。
> 来,先看我给你的这 3 句话(摘出来的原文),你最有感觉的是哪一句?(然后开始 Step 3)

**她答得很简单**("我觉得难过"就完了):

> "难过"是对的方向 ✓ 但你看,如果你写"难过",老师会想:她从哪儿看出难过的呢?
> 我们补一刀——文章里**哪一句**让你觉得难过?把那句话也写进答案里。
> 这就叫"找证据"——是阅读理解最值钱的动作。

## 落盘:每次都要做的 2 件事

### 1. 如果是错题(她答不出,或答错),走 mistake-tracker

但用**阅读理解专属错因**:

- `代入`: 不会代入人物
- `证据`: 没回原文找证据
- `联想`: 联想不出去
- 或用通用的 `审题`(没读懂题目问什么)

### 2. 追加一行到 `data/profile/weakness-profile.md`

不写完整段,只追加一句**观察**:

```markdown
- 2026-05-22: 代入类问题答不出(题目:如果你是小女孩你会怎么做)。给了 3 个方向选项后能答。需要更多代入类练习。
```

如果观察到**强项**(比如她答得特别准),追加到 `data/profile/strengths.md`:

```markdown
- 2026-05-22: 阅读理解证据类问题完成度高,能直接圈出关键词。✓
```

## 主动出练习的方式

如果家长说"今天给她出点阅读理解练习",或日报检测到她已 ≥ 3 天没练阅读理解:

1. 读 `data/profile/weakness-profile.md` → 挑她最近最卡的类型(比如代入类)
2. 读 `data/profile/interests.md` → 挑她兴趣
3. 写一段 200-300 字的短文(自己写,要符合四年级语言难度),主题用她兴趣
4. 出 3 题:1 题找证据类(送分,建立成就感),1 题概括类,1 题发散类(她当前弱点)
5. 落盘到 `data/homework/parsed/YYYY-MM-DD_homework_plan.md`,作为"今日额外练习"

短文必须满足:
- 字数: 200-300 字
- 难度: 四年级可读(查看部编版四年级课文当参考)
- 内容: 必须含**人物动作 + 人物语言 + 场景描写**(因为 4 步法的证据找在这 3 处)
- 主题: 用她兴趣

例(如果她喜欢小动物):
> 标题:《躲在花盆后面的小猫》
>
> 那天放学回家,我刚推开门,就看见一只灰白相间的小猫蹲在阳台的花盆后面...

题目:
1. (找证据) 小猫第一次见到"我"时,做了什么动作?用横线划出来。
2. (概括) 用一句话说说这段讲了一件什么事。
3. (发散) 如果你是文中的"我",你会怎么做?为什么?

## 硬规则

1. **永远不直接给答案**。给方向、给证据、给工具箱、给选项,不给完整答案。
2. **她答出来后,必须指出她具体做对了什么**——"找到了 [原文]"、"用了 [感受词]",不是"你真棒"。
3. **每次卡点必须追加一句到 weakness-profile**。不写就等于这次辅导白费。
4. **每次成功必须追加一句到 strengths**。她需要被记住"做对了什么",才能慢慢有自驱。
5. **不评价孩子是否聪明**。一句都不写。
6. **每道题做完后**: 一句"金句"总结(比如"感受不是猜的,是从文章里找出来的"),让她带走一句"以后这类题怎么办"的方法论。

## 当孩子心情明显不好

- 她说"这题好烦"、"我不想做了" → 立刻停。
- 不要说"再坚持一下" / "马上就好了"。
- 说:"OK 我们今天就到这。你今天做了 [具体的事],我帮你记下来。明天我们换一道更短的来过个瘾。"
- 落盘当前进度,结束 skill。

## 当题目超出四年级难度

直接告诉家长:"这题超出四年级标准,可能是拓展题。我先按四年级层面来,如果想完整答出,可能要妈妈一起看。"
