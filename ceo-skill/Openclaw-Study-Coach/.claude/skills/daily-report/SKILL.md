# 每日日报 Skill

## 什么时候触发

- 家长说"生成今日日报"、"看下今天怎么样"
- 孩子说"我做完了"、"今天结束了"
- 当天最后一项任务完成后

## 必读数据(按顺序读)

1. 当天 `data/homework/parsed/YYYY-MM-DD_homework_plan.md` — 看任务有几项,几项做完
2. 当天 `data/mistakes/review_queue/YYYY-MM-DD_queue.md` — 看新增几条错题
3. 当天 `data/points/log.jsonl` — 计算 `points_today`
4. `data/reports/trends/trends.csv` — 看 `streak_days` 的上一行值
5. `data/profile/weakness-profile.md` — 看当前 pattern,辅助"明日重点"选择
6. `data/profile/strengths.md` — 看本周做对的事,辅助"孩子一句话"
7. USER.md — 读 `kid_id`、`昵称`

## 日报必备 7 个 section

```markdown
# 2026-05-22 日报

**孩子**: meimei(小妹)
**学习时长**: 约 45 分钟
**积分**: +35(累计 312)
**连续学习**: 第 6 天 🔥

## 1. 今日完成

- ✅ 语文-阅读理解《小猫》3 题(2 对 1 卡)
- ✅ 语文-作文片段(写了开头一段,124 字)
- ✅ 数学-口算 P10(30/30 题)
- ⚠️ 英语-单词 10 个(只完成 7 个)

## 2. 今日突破(她做对的事,具体到瞬间)

> 这一节是 strengths.md 的当日新增,**孩子也能看**(在反馈她时调用)

- ⭐ 在《小猫》第 2 题找到了"轻轻地"做证据 — 这是阅读理解最关键的"找证据"动作
- ⭐ 作文里第一次自己用了感受词"忐忑" — 不是给的工具箱里的,是她自己想到的

## 3. 今日卡点(只写 1 个)

阅读理解《小猫》第 3 题"如果你是文中的小猫,你会怎么做?" — **代入类**发散题。给 3 个方向选项后才答出。

**错因分类**: 代入

**这是本周第 N 次同类卡点**: 第 3 次(参考 weakness-profile)

## 4. 错题新增

- 语文 - 阅读理解 - 代入(已加入 review queue,回流日期:2026-05-25)

## 5. 明日 1 个重点

**代入类阅读理解小练**——15 分钟。
- 我会出一段以小猫为主角的短文(用她兴趣)
- 配 3 题:1 题找证据(送分),1 题概括,1 题代入(继续这周的弱点)

## 6. 给家长一句话(可执行)

小妹今天做了 45 分钟,作文有明显进步(第一次自己用了"忐忑")。代入类阅读题是这周第 3 次,需要继续练但不焦虑——给方向选项就能答了,说明她不是不会,是没掌握"先想方向再答题"这个思路。

## 7. 给小妹一句话(她明天会看到)

小妹今天闯到第 6 关 🎉 作文里你用了"忐忑"这个词 — 这个词我以前没教过你,你自己想到的,真的很厉害。明天我们继续闯。

## 8. 今晚父母可以做的 1 件事(3 分钟)

(parent-playbook 自动生成,详见 `data/parent-playbook/daily/2026-05-22_playbook.md`)

> **晚饭时妈妈可以问小妹**:
> "今天那只小猫的故事里,小猫最后躲在花盆后面没出来。**如果家里有一只小猫这样躲起来,妈妈会怎么做?**"(故意先给妈妈的版本)

>

```

## 落盘:6 个动作(必须**全部**完成)

### 1. 写日报文件

路径: `data/reports/daily/YYYY-MM-DD_daily.md`
内容: 上面 8 个 section 的完整 markdown(包括父母 playbook 那一节)

### 2. 追加 trends.csv 一行

格式:`date,kid,homework_minutes,mistakes_new,main_subject,top_knowledge,top_reason,points_today,streak_days`

例:
```csv
2026-05-22,meimei,45,1,语文,代入类阅读理解,代入,35,6
```

### 3. 更新 USER.md streak

读 USER.md 找到:
```
- 连续学习天数: 5
- 最后一次学习日期: 2026-05-21
```
如果"最后日期"是昨天 → streak +1
如果更早 → streak 重置为 1
更新这两行。

### 4. 即时更新 weakness-profile.md

追加单行到"已观察但暂不归 pattern"区:
```markdown
- 2026-05-22: 代入类阅读题答不出,给 3 方向选项后答出(《小猫》第 3 题)
```

如果同一类卡点本周已累计 ≥ 3 次,把这条升级到"当前关注 pattern"区,带 4 字段(表现/频次/已尝试/下一步)。

### 5. 即时更新 strengths.md

把当日突破追加到"最近做得好的具体瞬间"区:
```markdown
- 2026-05-22: 在《小猫》第 2 题找到"轻轻地"做证据 ⭐
- 2026-05-22: 作文里自己用"忐忑"这个词 ⭐
```

### 6. 调用 parent-playbook 生成今晚活动

按 parent-playbook SKILL.md 的逻辑,根据当天数据选模板,写到 `data/parent-playbook/daily/YYYY-MM-DD_playbook.md`。

**6 个动作做完后**,在对话里逐一公示:"已写入 X / 已追加 Y / 已生成 Z"。

## streak 计算规则

- 当天有 ≥ 1 项任务完成 → 算 1 天学习
- "完成"以 `data/points/log.jsonl` 当天至少 1 条 delta > 0 为准
- 周末和节假日也计入
- streak 重置: 中断 ≥ 1 天 → 下次再开始时归 1
- 上限不封顶,反馈里超过 30 显示 "30+"

## 字段填法

| 字段 | 怎么填 |
|---|---|
| **今日完成** | 逐项列任务,✅/⚠️/❌ 状态。 |
| **今日突破** | 1-3 条她做对的具体瞬间。是 strengths.md 的当日新增。如果没有,**主动找一个**——哪怕只是"今天坚持做了 45 分钟"。 |
| **今日卡点(1 个)** | **只写 1 个**,选影响最大的。多了家长盯不动。 |
| **错题新增** | 数当天 queue 文件里新加的条目数。分学科列出。 |
| **明日 1 个重点** | 必须可执行,带时长。优先从 weakness-profile.md 的"当前关注 pattern"里选。 |
| **给家长一句话** | 不指责,可执行,≤ 50 字。 |
| **给孩子一句话** | 鼓励 + 下一步。**不写"加油"、"棒"这种空话**。必须指出**具体做对**了什么。 |
| **今晚父母可做的 1 件事** | parent-playbook 生成,3 分钟可做。 |

## 当天孩子什么都没做

- 还是要生成日报
- "今日完成": 0 项
- "今日突破": "今天她休息了一天,这本身也是节奏的一部分"(不夸张说她有进步,但也不批评)
- streak: 维持(还没断)或归 0(已经断 2 天)
- 父母 playbook: 类 D(纯陪伴型)
- trends.csv: `2026-05-22,meimei,0,0,,,,0,0`

## 当天孩子情绪化中断

- 日报里如实记录,语气不带评价
- "今日完成"标 ⏸️(暂停)
- "今日卡点": "今天的卡点不是知识,是状态"(不深入)
- 父母 playbook: **强制类 D**(陪伴型,不学习)

## 重复生成同一天日报

追加新 section,不要覆盖:

```markdown
## 补充更新 (19:30)

(新内容)
```

## 不要做

- ❌ 不把日报发到任何外部
- ❌ 不在日报里出现真实姓名 / 学校(代号化)
- ❌ 不写"再接再厉"、"持之以恒"这种空话
- ❌ 不在"今日突破"里写"她有进步"这种判断性的话,只写**具体瞬间**
- ❌ 不在父母 playbook 里给抽象建议(必须是具体台词/动作)
