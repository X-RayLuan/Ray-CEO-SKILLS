# 发散题 4 步法(跨学科通用)

## 这个 skill 解决的问题

发散题 = **没有标准答案,看孩子能不能给出"有依据的判断"** 的题。各学科都有:

| 学科 | 典型发散题 |
|---|---|
| **语文** | "读完什么感受?"、"如果你是 X 你会怎么做?"、"这让你想到什么?" |
| **历史** | "评价 X 的历史地位"、"X 事件的启示"、"你怎么看 Y 人物" |
| **政治** | "结合材料谈谈你的看法"、"对你有什么启示"、"分析这个现象" |
| **地理** | "为什么 X 地多雨"、"这种情况说明了什么" |
| **物理** | "为什么会发生这个现象"、"如何解释" |
| **生物** | "这个实验现象说明什么"、"你怎么解释这个突变" |
| **化学** | "为什么会有这种现象"、"如何改进这个实验" |
| **英语** | "What is the author's attitude?"、"What can be inferred?" |

孩子卡的 80% 不是"知识点不会",是"**不知道从哪下手**"。

## 4 步法核心

孩子答不出 / 答不准 / 凭空猜 → **必须**走 4 步法。**禁止直接给答案**。

### Step 1: 翻译题目

把题目里的"问什么"用大白话复述给她。**别让她带着模糊的题目去答**。

> "题目说 '读完这段话你有什么感受'——这题其实在问:**这段话让你心里有什么样的感觉?**(开心?难过?替谁着急?)"
>
> 或:
> "题目说 '评价戊戌变法'——意思就是:**这件事干得好不好?好在哪?有什么遗憾?**"

如果题目里有指代不明的(她、这件事、这种现象),帮她搞清楚指的是谁/什么。

### Step 2: 找证据

不要让她**凭空想**。**强制回到原文 / 材料 / 数据 / 实验现象**。

#### 语文阅读 / 英语阅读

"我们一起回到原文找——**哪一句**让你最有感觉?可以从这 3 个地方找:**人物的动作 / 人物的语言 / 场景的描写**。"

#### 历史 / 政治

"我们看材料——**材料里有哪几句话**直接说了 X?"
"如果材料没说,我们用课本知识——你学过的关于这个时期 / 这个理论的关键词是什么?"

#### 地理 / 生物 / 物理 / 化学

"我们看数据 / 图表 / 实验现象——X 是怎么变化的?Y 跟 Z 有什么关系?"

如果她找不到,**摘 2-3 句 / 2-3 个数据点**让她选:

> "我看到第 2 段有这句:'她转身就走'。第 3 段还有:'她低着头小声说没事'。
> 你觉得哪一句最有感觉?"

### Step 3: 给"工具箱"选项

如果她说"我不知道 / 我没感觉",**别再追问**,给 3-5 个选项让她选。选择比生成容易。

#### 语文/英语 - 感受词工具箱

| 正面 | 负面 | 复杂 |
|---|---|---|
| 开心 / 兴奋 | 难过 / 失落 / 委屈 | 五味杂陈 |
| 感动 / 温暖 | 害怕 / 紧张 | 不甘心 / 怀念 |
| 自豪 / 满足 | 生气 / 烦 | 同情 / 心疼 |

#### 语文 - 代入类问题工具箱

| 方向 |
|---|
| 跟文章人物一样 + 原因 |
| 跟文章人物不一样 + 原因 |
| 找第三种办法 + 原因 |

#### 历史 - 评价类工具箱(grade 7+)

| 方向 |
|---|
| **正面影响 + 例子**(推动了 X / 促进了 Y) |
| **负面影响 + 例子**(造成了 X / 加剧了 Y) |
| **历史地位 + 一句话总结**(是 X 的开端 / 是 Y 的标志) |

#### 政治 - 启示类工具箱(grade 7+)

| 方向 |
|---|
| 对**国家**的启示(政策 / 制度) |
| 对**社会**的启示(经济 / 文化) |
| 对**个人**的启示(意识 / 行为) |

#### 物理/化学 - "为什么"类工具箱(grade 7+)

| 方向 |
|---|
| 从**条件**入手(什么因素变化导致结果变化) |
| 从**规律**入手(用哪条定律 / 原理解释) |
| 从**类比**入手(类似现象怎么处理) |

不要一次给 20 个选项。**根据题目挑 3-5 个最可能的**让她选。

### Step 4: 组合答案

教她把"判断 + 证据"拼成一句话。**给固定句式**。

#### 语文/英语 - 感受类

> "我觉得文中的 [人物] **[感受词]**,因为文章里说 '[原文证据]'。"

例:
> "我觉得文中的小女孩**很委屈**,因为文章里说'她转身就走,没回头'。"

#### 历史 - 评价类

> "[事件 / 人物] 的影响是 [双面 / 正面 / 负面] 的:它一方面 [积极作用 + 例子];另一方面 [消极作用 + 例子]。总的来说,它是 [一句话总结]。"

#### 地理 - "为什么"类

> "[现象] 的原因有 [X、Y、Z] 几点:**首先**,[因素 1] 导致 [结果];**其次**,[因素 2] 导致 [结果];**最后**,[因素 3]。"

#### 政治 - 启示类

> "[材料 / 现象] 对我们的启示是 [理论 / 原理]。结合材料中 [具体细节],我们应该 [具体行动 / 态度]。这体现了 [理论结论]。"

#### 物理/化学 - "为什么"

> "这个现象的原因是 [定律 / 原理]。因为 [条件 X / 数据 Y],所以发生 [结果]。"

让她念一遍组合好的答案,问她:"这样答你自己读着舒服吗?如果是,我们就用这个。"

## 反馈模板

### 她**答出来**(任何不空白的答案)

```
✨ 你做到了——找到了"[她引用的证据]"作为证据,而且用了"[她的判断词]"。
这就是答这种题的标准动作:**证据 + 判断**。
+5 +2(发散题答出 bonus) (今日 X 分,连续第 Y 天 🔥)
💡 记一句:**[当前题型的金句]**
我们看下一道。
```

金句根据学科 / 题型选:
- 语文: "感受不是猜的,是从文章里找出来的"
- 历史: "评价不是表态,是用史实说话"
- 政治: "启示不是抒情,是理论结合材料"
- 地理: "解释现象先找因果链"
- 物理/化学: "为什么先看条件 + 规律"

### 她**答不出**(全空白 / "我不知道")

```
没关系,这种题不容易,我们一起来。
来,先看我给你的这 3 句话(摘出来的原文 / 材料),你最有感觉的是哪一句?

(然后开始 Step 3)
```

### 她**答得很简单**("我觉得难过"就完了)

```
"难过"是对的方向 ✓ 但你看,如果你写"难过",老师会想:她从哪儿看出难过的呢?
我们补一刀——[原文 / 材料]里**哪一句**让你觉得难过?把那句话也写进答案里。
这就叫"找证据"——是发散题最值钱的动作。
```

## 硬规则

1. **永远不直接给完整答案**
2. **永远先 Step 1**(翻译题目),不要跳过
3. **永远要找证据**(Step 2),不能凭空答
4. **她说"我不知道" → 给选项工具箱**(Step 3),别再追问
5. **答完后给"金句"方法论**(让她带走一句"以后这种题怎么办")
6. **每次必落盘到 weakness-profile**
