# 父母 playbook Skill

## 这个 skill 解决的问题

沈奕斐(教育学者)说,培养孩子自驱力,要让孩子对学习有兴趣;父母要把学科变成游戏。

但父母自己**想不出来怎么玩**——这是真实的痛点。

这个 skill 把"想小游戏"的工作从父母身上挪到 agent 身上。每天日报后,生成一份**今晚 3 分钟可做的**亲子活动;每周五生成一份**周末活动**。

## 什么时候触发

1. **daily-report 收尾时**自动触发,生成 `data/parent-playbook/daily/YYYY-MM-DD_playbook.md`
2. **weekly-report 在周五跑完时**触发,生成 `data/parent-playbook/weekend/YYYY-Www_weekend.md`
3. **家长说**"今晚做点什么?"、"周末做点什么?" 直接触发

## 日活动必备 5 条原则(每条都要满足)

1. **3 分钟内能开始**——不打印、不买东西、不开电脑、不在网上下载素材
2. **不像作业**——是聊天、是游戏、是顺嘴一问,不是"你来做一题给我看"
3. **接今天的学习**——延伸今天她做的题 / 写的字 / 读的故事,**不凭空起新内容**
4. **AI 做不了的**——真人陪伴 / 实物互动 / 情绪共鸣 / 生活观察 / 家庭仪式感
5. **给到位的具体**——具体到台词级别("妈妈说:'XXX'")或动作级别,不给"建议加强 XX"

## 日活动 4 类模板(轮换出)

### 类 A:把今天的语文延伸成晚餐对话

如果今天孩子做了阅读理解,问父母:

> **今晚妈妈可以做的 1 件事**(3 分钟):
>
> 晚饭时,问小妹一个问题:
>
> > 你今天看的那篇《XX》里,小女孩 [今天文章里发生的具体动作]。**如果是我们家,会怎么样?**
>
> 让她答。她答完之后,你**说一个跟她不一样的版本**(故意夸张一点),看她反应。
>
> 💡 这个动作在练她今天在阅读理解里卡的"代入想象"——而且让她看到大人也会有不同的想法,降低她答题时"我必须答对"的压力。

### 类 B:把今天的作文细节延伸成"实物对照"

如果今天孩子写了作文,挑她作文里的一个具体细节,让父母用实物对照:

> **今晚爸爸可以做的 1 件事**(2 分钟):
>
> 看小妹今天作文里写了"妈妈穿了一件蓝色的裙子"。
>
> 1. 找一个家里**真正蓝色**的东西(妈妈的衣服 / 一个杯子 / 窗帘 / 天空也行)
> 2. 问她:"小妹,你说这种蓝叫什么蓝?天蓝?海蓝?牛仔蓝?还是别的?"
> 3. 听她回答。然后告诉她:"以后你作文里写蓝色,可以更具体——'像天空那样的蓝'就比'蓝色'有画面多了"。
>
> 💡 这个动作在帮她把"颜色词"变成她真正能调用的词,而不是"蓝色 / 红色"这种笼统词。

### 类 C:"角色互换"——让孩子当一次小老师

如果今天孩子在某道题上有突破:

> **今晚妈妈可以做的 1 件事**(5 分钟):
>
> 今天小妹在阅读理解第 3 题上做对了——她找到了"轻轻地"做证据。
>
> 把这个动作当成一个"她教你"的机会:
>
> 1. 把那道题原文复印一份(或者直接念给她听,假装你不会)
> 2. 跟她说:"妈妈这题真不会,你能教我一下吗?"
> 3. 让她**给你讲**怎么找证据,怎么答
> 4. 听她讲完,你假装"原来这样,我懂了"
>
> 💡 让她当一次老师,是她最容易获得"我会"的成就感的方式。也帮你看出她**真的理解**了多少。

### 类 D:轻松型——纯陪伴,不学习

如果今天孩子明显累了 / 心情不好 / 已经 streak ≥ 5 天:

> **今晚妈妈可以做的 1 件事**(3 分钟):
>
> 今天小妹完成了 [具体什么],已经连续学习第 6 天了。今晚**不做任何学习相关的事**。
>
> 试试这个:
>
> 1. 睡前问她:"今天你最开心的一件事是什么?"(**注意:不问学习的事**)
> 2. 她答完之后,你也说**你今天最开心的一件事**
> 3. 不评价、不引申、不教育。就单纯交换两个一天里开心的瞬间
>
> 💡 自驱力的底层不是"学得多",是"和爸爸妈妈关系好"。这一晚是给系统充能,不是耗能。

## playbook 文件格式

`data/parent-playbook/daily/YYYY-MM-DD_playbook.md`:

```markdown
# 2026-05-22 父母 playbook

**触发原因**: 今天小妹做了 2 道阅读理解,其中代入类卡了
**类型**: 类 A(延伸成晚餐对话)
**预计时长**: 3 分钟
**适合谁做**: 妈妈(因为晚饭通常妈妈在)

## 今晚妈妈可以做的 1 件事

(详见上面 类 A 模板的具体内容,带具体台词)

## 这件事在练什么

她今天在阅读理解的"代入"上卡住了,这个晚餐对话延伸了"代入"练习,但**用游戏感包装**——不是再做一道题。

## 备选方案(如果今晚妈妈没空)

(给出 1 个 2 分钟内更短的版本,或可以推到周末的)

## 完成后你可以问 agent 什么

如果今晚做了,明天早上跟 agent 说:
> "昨晚我和小妹玩了那个对话游戏,她答了 XX,我答了 YY,看起来很开心。"

agent 会:
- 把这个家庭活动效果写进 strengths.md(让她下次看到自己又一次的成就)
- 在今天的练习里调整难度(她已经在生活场景里练过了,可以直接上稍难一点的)
- 写进 weakness-profile 的"已尝试方案"

---
*生成时间: 2026-05-22 21:00 | 数据来源: daily report + weakness-profile + interests*
```

## 周末活动(每周五由 weekly-report 触发)

更深一点的活动,15-30 分钟,适合周末整段时间。

`data/parent-playbook/weekend/YYYY-Www_weekend.md`:

### 周末活动 4 类模板

#### 类 1:阅读类(亲子共读)

读一篇短文 / 故事 / 绘本,**但读完不做题**。用 2 个问题聊:
- "你最喜欢这个故事的哪一段?"
- "如果你是 XX,你会怎么做?"

#### 类 2:写作类(家庭小报)

让她当一周一次的"家庭记者",写 100 字记录这一周家里发生了什么。
- 父母只问问题不写
- 写完贴在冰箱上 / 收进一个文件夹,以后做"小妹作文集"

#### 类 3:观察类(自然/城市/家)

带她去一个地方(楼下花园 / 菜市场 / 公园),只做一件事:**观察并记录 3 个细节**。
- 1 个看到的(颜色 / 形状)
- 1 个听到的(声音)
- 1 个感觉到的(温度 / 心情)

回家后她讲给你听,你帮她**用 3 句话写下来**。这就是下周一作文素材。

#### 类 4:游戏类(语言游戏)

不需要道具的语言游戏。3 个推荐:

- **"我说一个词,你接一个有画面的词"**(妈妈说"红"→ 小妹说"消防车"/ "番茄"/ "枫叶")
- **"形容词接力"**(妈妈说"妈妈很 XX",小妹接"小妹很 XX")
- **"故事接力"**(轮流每人说 1 句,把故事接下去)

## 周末 playbook 文件格式

```markdown
# 2026-W21 周末活动 (5/24-5/25)

**类型**: 类 2(家庭小报)
**为什么这周选这个**: 小妹这周作文卡在开头,改成"记录"任务可以避开"作文"这个标签
**预计时长**: 30 分钟
**最佳时段**: 周日上午

## 完整指南

(详细的、可执行的步骤,以及具体台词)

## 这周的具体"采访问题"清单

(给爸爸/妈妈 5 个具体问题,让他们问小妹)

1. 这周哪一天最特别?为什么?
2. 这周有没有发生什么你想跟我说的事?
3. ...

## 完成后

把小妹写的家庭小报拍照发给 agent,agent 会:
- 把这次写作存档(`data/homework/done/...`)
- 提炼出她这次用到的 1-2 个好的写法,写进 strengths.md
- 这个素材下次出作文题时直接用
```

## 选模板的判断逻辑

每天 daily-report 跑完后,parent-playbook 这样决策:

```
1. 看今天 streak_days
   - 如果 ≥ 5 → 优先类 D(陪伴型,给孩子充电)
   
2. 看今天 main_subject 和 top_reason
   - 阅读理解卡了 → 类 A(晚餐对话延伸)
   - 作文写了 → 类 B(实物对照细节)
   
3. 看今天有没有突破(strengths.md 当天新增 ≥ 1 条)
   - 有 → 类 C(角色互换)
   
4. 没明显信号 → 默认类 A
```

## 周五额外触发周末活动

每周五生成 weekend playbook 时,看本周整体情况:

- 本周作文有突破 → 类 2(家庭小报)
- 本周一直卡阅读理解 → 类 1(共读)
- 本周状态低 → 类 3(观察类,出门换状态)
- 本周状态高 → 类 4(语言游戏,把高状态延续)

## 硬规则

1. **每天的 playbook ≤ 5 分钟**。多了父母不会做。
2. **必须接今天的内容**——参考当天 daily-report、当天 strengths/weakness 新增。
3. **永远不要"建议加强 XX"这种空话**。必须给到台词级别 / 动作级别。
4. **永远不要给学习负担**。playbook 是亲子时间,不是补习。
5. **如果今天孩子明显累 / 心情不好 → 类 D**(纯陪伴),不论今天学了什么。
6. **每天写一份新的**,不要复用昨天的。父母会很快烦同一个活动。
7. **每份 playbook 最后必须有"完成后跟 agent 反馈"的链路**。父母做完了告诉 agent,agent 学到东西,下次出得更准——这是循环的关键。
