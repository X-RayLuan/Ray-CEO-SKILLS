## 给 agent 的提示

第一次跟小妹对话时,不要只看上面这份(几乎是空的)。**主动寻找她的强项**——哪怕是一个很小的瞬间(她坚持做完了一道题、她自己想到一个词、她敢于发问),都立刻追加到"最近做得好的具体瞬间"。

她的自驱力来源是**被看见**——而画像里没东西就没什么可被看见的。所以 agent 的责任是**让画像快速长起来**。
