## 给 agent 的提示

第一周对话里,**有意识地**从孩子日常话题里挖兴趣(不要直接审讯式问她"你喜欢什么"——10 岁孩子可能答不出)。从这些信号挖:

- 她写作文时自然写到的东西
- 她做阅读理解时联想到的东西
- 她拒绝某个故事题材时的原因
- 她主动提到的人 / 事 / 物

把这些信号即时追加到这里。**1 个月后,这份文件应该比 weakness-profile 长 3 倍**——孩子的兴趣是这套系统最值钱的资产。
