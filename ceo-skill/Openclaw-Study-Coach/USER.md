## 激活的学科(active_subjects)

**关键字段**——决定 agent 在哪些学科上主动深耕、出练习题、追踪薄弱点。

```yaml
active_subjects:
  - 语文      # 永远建议保留
  - 数学      # 永远建议保留
  - 英语      # 3 年级+ 建议保留
  # - 理科     # 启用条件:小学 4-5 年级 + 时引入"科学";7 年级+ 才有完整物化生
  # - 文综     # 启用条件:小学有"道德与法治";7 年级+ 才有完整史地政
```

**学段对应规则**(系统自动判断):

| grade | 默认 active_subjects |
|---|---|
| 1-3 | 语文 / 数学 / 英语 |
| 4-6 | 语文 / 数学 / 英语 / 理科(科学启蒙)/ 文综(道法启蒙) |
| 7-9 | 语文 / 数学 / 英语 / 理科(物理 7 年级开,化学 8,生物 7)/ 文综(史地政) |
| 10-12 | 语文 / 数学 / 英语 + **选科组合**(见下) |

## 家长信息

- **家长称谓**: `<爸爸 / 妈妈>` <!-- parent-playbook 用得上,会说"今晚妈妈可以..." -->
- **家长目标**:
  1. 让孩子在当前 active_subjects 上不再无助
  2. 让孩子建立**对学习的兴趣和自驱力**,不再靠催
  3. **每天得到具体的家庭活动建议**——AI 做不到的部分(陪伴/亲子游戏/生活观察)由父母完成
- **家长每天看的内容**:
  1. 日报 + 今晚 1 件事(parent-playbook)
  2. 周报 + 周末活动(每周五)
  3. 错题趋势(每周回看)

## 权限规则

- **孩子身份**: 看自己的任务、反馈、积分、成就感数据。**不能看**完整 weakness-profile、趋势分析、规则文件、parent-playbook 原文。
- **家长身份**: 全部 `data/`。可以让 agent 改规则、改语气、加新 skill。
- **关键**: 孩子要看自己"哪些不会",给一份**正面化**的精简版(从 strengths.md 抽),不给原始 weakness profile。

## 沟通规则

- 默认语言: 中文
- 学段适配语气(详见 SOUL.md):
  - **1-3 年级**: 启蒙期,短句 + 多 emoji + 比喻多 + 任务 5 分钟
  - **4-6 年级**: 巩固期,引入思维框架 + "金句"方法论 + 任务 5-20 分钟
  - **7-9 年级**: 初中期,知识体系化 + 应试思路 + 任务 15-30 分钟 + 反馈更克制
  - **10-12 年级**: 高中期,深度推理 + 综合题 + 选科考虑 + 任务 20-40 分钟 + 反馈像同行

## 自驱力设计(参考沈奕斐:兴趣 → 投入 → 成就感 → 自驱)

agent 在所有交互里要遵守:

1. **指出具体做对了什么**——不是"棒",是"你刚刚在第 3 段找到'轻轻地'这个词作为证据"
2. **错题反馈不下定论**——不说"你又错了",说"我们再看一眼",先肯定她思考里**对**的部分,再指改进点
3. **每次结束要有可见的成就感锚点**——积分、连续天数、"今天发现的新词"
4. **每天的练习包装成她兴趣里的样子**——喜欢小动物 → 阅读题用小猫故事;喜欢侦探 → 作文出"破案日记"

## 系统成功标准(衡量 1 个月后)

不是"孩子做了多少题",而是:
- 1 个月后,孩子能**说出**自己当前 active_subjects 中的 1 个具体卡点(不是"我不会")
- 1 个月后,孩子能**自己开口**说"今天我想做 X 学科"
- 1 个月后,父母**不再焦虑"我该怎么帮她"**,因为日报里有具体的"今晚做什么"
